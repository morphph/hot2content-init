# Qwen 3.5本地部署

**一句话总结：** 单张 RTX 3090 即可跑满 262K 上下文的 Qwen3.5-35B-A3B，4-bit 量化下达到 113 tokens/秒，本地部署门槛大幅降低。

## 为什么现在值得关注

Qwen 3.5 刚刚登陆 Ollama（v0.17.4），这意味着一行命令就能在本地跑起来。更重要的是，社区实测数据相当惊艳：一位开发者用单张 RTX 3090（24GB 显存）运行 Qwen3.5-35B-A3B 的 4-bit 量化版本，在完整 262K 上下文窗口下跑出了 113 tokens/秒的推理速度。这个数字意味着你可以抛开 API 订阅和代理，直接在本地接入 Claude Code 等工具链。

## 硬件需求

| 配置档位 | 显存要求 | 推荐显卡 | 适用场景 |
|---------|---------|---------|---------|
| 入门 | 8GB | RTX 4060/3060 | 小模型、短上下文 |
| 主流 | 16GB | RTX 5080/4080 | Q4_K_M 量化、中等上下文 |
| 进阶 | 24GB | RTX 3090/4090 | 全量 262K 上下文 |

Reddit 社区的后续测试表明，RTX 5080（16GB）使用 Q4_K_M 量化配合 `--fit on` 参数可以达到 74.7 tok/s，比默认配置提升约 7%。
