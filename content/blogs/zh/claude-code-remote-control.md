# Claude Code 远程控制功能

**一句话总结：** 在终端启动 Claude Code 任务后，用手机随时随地查看进度、审批操作、继续对话——代码在本地跑，你人在外面浪。

## 这功能解决什么问题

写代码时最纠结的场景之一：Claude 正在帮你跑一个耗时任务，但你需要离开电脑。以前只能干等着，或者冒着任务中断的风险走开。

Remote Control（远程控制）功能让你摆脱这个困境。任务继续在你的本地机器上执行，而你可以通过手机上的 Claude 应用或 claude.ai 网页端实时监控和操作。

正如 Anthropic 官方推文所说：

> Start local sessions from the terminal, then continue them from your phone. Take a walk, see the sun, walk your dog without losing your flow.

## 谁能用

目前该功能正在向 Max 订阅用户逐步开放，处于 research preview 阶段。如果你是 Max 用户，更新到最新版 Claude Code 即可尝试。
