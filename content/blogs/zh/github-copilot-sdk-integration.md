# GitHub Copilot SDK 集成指南

**一句话总结：** GitHub 官方发布多平台 SDK，让开发者能将 Copilot Agent 能力嵌入自己的应用和服务中。

## 背景

GitHub 近期在 Trending 榜单上线了 `github/copilot-sdk` 项目，这是一套多平台 SDK（Software Development Kit），允许开发者将 GitHub Copilot Agent 的能力集成到自有应用和服务中。这意味着 Copilot 不再局限于 VS Code 或 GitHub 网页，而是可以作为底层能力被任何产品调用。
