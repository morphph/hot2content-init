## 🎓 MODEL LITERACY: Mixture of Experts (MoE)

Several trending models this week use **MoE (Mixture of Experts)** architecture — like NVIDIA's Nemotron-Cascade-2-30B-A3B and various Qwen3.5 variants.

**What is MoE?** Instead of using the entire model for every input, MoE routes each token through only a subset of "expert" sub-networks. A gating mechanism decides which experts handle which inputs. This means a 30B parameter model might only activate 3B parameters per token — giving you near-big-model quality at near-small-model speed and cost.

**Why it matters:** MoE is how the industry is making frontier-quality models affordable. When you see "30B-A3B" in a model name, it means 30B total parameters but only 3B active at a time.
