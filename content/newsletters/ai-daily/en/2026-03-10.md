## 🎓 MODEL LITERACY: What Is "Mixture of Experts" (MoE)?

You'll see "MoE" everywhere in today's model releases (Qwen3.5-397B-A17B, Qwen3.5-35B-A3B). Here's what it means:

A standard LLM runs every parameter for every token. A Mixture of Experts model splits its parameters into groups ("experts") and only activates a small subset for each token. So Qwen3.5-397B-A17B has 397 billion total parameters but only uses ~17 billion per token — giving you the knowledge of a huge model at the speed of a small one.

**Why it matters:** MoE makes it possible to run very capable models on less hardware. The tradeoff is higher memory requirements (all experts need to be loaded), but inference is fast because only a fraction does the work.
