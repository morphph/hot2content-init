## 🎓 MODEL LITERACY: Git Worktrees

**What is a git worktree?**
Normally, a git repo has one working directory. A worktree lets you check out multiple branches simultaneously in separate directories — all sharing the same `.git` history.

**Why it matters for AI coding:**
When you run multiple AI agents on the same repo, they'd normally conflict (editing the same files, fighting over git state). With worktrees, each agent gets its own directory and branch. They work independently, then you merge results.

Think of it like giving each AI coder their own desk in the same office, instead of making them all share one keyboard.

**Try it:** `git worktree add ../my-feature feature-branch`
