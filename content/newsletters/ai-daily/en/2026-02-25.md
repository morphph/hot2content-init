## 🎓 MODEL LITERACY: Mixture of Experts (MoE)

Qwen3.5-397B-A17B has 397 billion total parameters but only activates 17 billion per token. This is **Mixture of Experts (MoE)**. Instead of running every parameter for every input, the model routes each token to a small subset of specialized "expert" networks. Result: you get the knowledge capacity of a 397B model at roughly the compute cost of a 17B model. That's why the 35B-A3B variant can run on a single GPU—it only activates 3B parameters at a time despite having 35B total.

## 🎯 PICK OF THE DAY: Claude Code Remote Control

This changes the developer workflow in a fundamental way. Coding agents currently chain you to your desk—you need to monitor, approve permissions, review changes. Remote Control breaks that chain. Start a complex refactor, walk to lunch, approve a permission request from your phone, review the diff on the bus.

The real implication: **asynchronous human-AI collaboration**. The agent works continuously; the human checks in when convenient. This is the direction all coding agents will go—Codex already runs in the cloud, but Anthropic's approach keeps the session local (your machine, your filesystem) while making the control layer mobile. Expect every major coding tool to ship something similar within months.
