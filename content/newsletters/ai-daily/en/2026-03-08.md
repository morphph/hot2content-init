## 🎓 MODEL LITERACY: What is Tool Search?

Traditional agent architectures pass all available tools to the model in every request — even if most won't be used. This wastes tokens and can confuse the model. **Tool Search** (introduced by OpenAI in GPT-5.4) lets the model dynamically discover relevant tools at runtime, similar to how RAG retrieves relevant documents. The model describes what it needs, and the system surfaces matching tools. This is crucial for agents with dozens or hundreds of tools — it keeps context clean and costs down.
