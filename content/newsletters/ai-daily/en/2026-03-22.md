## 🎓 MODEL LITERACY: What Is an Agent Harness?

You've probably seen "agent harness" in multiple items today. An agent harness is the scaffolding around an LLM that turns it from a chatbot into an agent — it manages the loop of thinking, acting, observing results, and thinking again.

The harness handles everything the model can't: calling tools, managing memory across context windows, retrying on failure, and deciding when to stop. Think of the LLM as the brain and the harness as the nervous system plus skeleton. A great model in a bad harness will underperform a decent model in a well-engineered one. This is why Anthropic's post on "effective harnesses for long-running agents" matters — the harness, not the model, is increasingly the bottleneck for agent reliability.
