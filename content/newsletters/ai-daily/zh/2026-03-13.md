## 🎓 概念科普：混合专家模型（Mixture of Experts, MoE）

**是什么：** MoE 模型里有很多"专家"子网络，但每次推理只激活其中几个。就像去医院，有心内科、神经科等各种专家，但你只看跟自己病情相关的科室。

**为什么现在很火：** Qwen3.5-35B-A3B 就是典型例子——"35B"是总参数量，"A3B"表示每个 token 只激活约 30 亿参数。效果接近 350 亿，推理成本接近 30 亿。所以它下载量暴涨——在消费级 GPU 上跑得飞快，性能还超出预期。

**代价是什么：** 所有专家都要加载到显存里（占用多），但每个 token 的计算量少。推理快，但对显存小的设备不太友好。

## 🎯 今日精选：Claude Code 的 /loop——持久化 Agent 时代来了

Claude Code 的 `/loop` 命令是这个月最值得关注的 agent 功能之一。不再是"给个任务跑一次"的模式，而是可以设置最长 3 天的循环任务。

意味着什么：持续监控 PR、每日 Slack 摘要、自动检测 build 失败并修复。Claude Code 正在从"你调用的工具"变成"跟你一起工作的 agent"。再加上新的 Code Review（每个 PR 派 agent 团队审查），Anthropic 正在打造全天候的工程 agent——处理软件开发中繁琐的部分，让人专注于架构和设计决策。
