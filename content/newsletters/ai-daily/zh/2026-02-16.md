## 🎓 模型小课堂：Skills 与 Agent 规范

**什么是编码 agent 中的"Skills"？** OpenAI（Codex）和 Anthropic（Claude Code）现在都支持"Skills"——可复用、可分享的指令集，告诉编码 agent *如何*执行特定任务。把它们想象成插件或菜谱：不用每次都解释项目配置，写一次 SKILL.md，agent 就会照做。OpenAI 刚在 Responses API 中推出 Skills（2/10），支持本地和托管执行。这与 Claude Code 现有的 AgentSkills 系统相呼应。趋势已经清晰：跨 agent 的可移植专业能力正在成为标准模式。

## 🎯 今日之选：Agent 编码评测中的基础设施噪声

Anthropic 工程团队的深度调查表明，**仅基础设施配置就能使 benchmark 结果波动数个百分点**——有时比排行榜上"第一名"和"第三名"之间的实际差距还大。这很重要，因为整个行业正在基于这些 benchmark 做数十亿美元级的决策。如果你的 Docker 配置或网络超时设置对分数的影响大于模型质量本身，那我们到底在衡量什么？对构建、评估或采购 AI 编码工具的人来说，这是必读文章。
