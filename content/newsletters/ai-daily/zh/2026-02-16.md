## 🎓 概念科普：Skills 与 Agent 规范

**编码 Agent 中的 "Skills" 是什么？** OpenAI（Codex）和 Anthropic（Claude Code）现在都支持 "Skills"——可复用、可共享的指令集，告诉编码 Agent **如何**执行特定任务。可以理解为插件或食谱：不用每次解释你的项目配置，写一次 SKILL.md，Agent 就会遵循。

OpenAI 刚刚在 Responses API 中推出了 Skills（2 月 10 日），支持本地和托管执行。这与 Claude Code 现有的 AgentSkills 系统一致。两家殊途同归意味着：**跨 Agent 可移植的专业能力正在成为标准模式**。
