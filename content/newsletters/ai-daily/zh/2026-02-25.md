## 🎓 概念科普：混合专家模型（Mixture of Experts, MoE）

Qwen3.5-397B-A17B 总参数 397B，但每个 token 只激活 17B。这就是 **MoE（混合专家模型）**。模型不是让所有参数都参与计算，而是把每个 token 路由到一小组专门的「专家」网络。效果：拥有 397B 模型的知识容量，但计算成本接近 17B 模型。所以 35B-A3B 版本能在单张显卡上跑——每次只激活 3B 参数，虽然总共有 35B。

## 🎯 今日精选：Claude Code 远程控制

这从根本上改变了开发者的工作方式。现在的编程 agent 把你钉在桌前——要监控、要批准权限、要审查改动。Remote Control 打破了这个束缚：启动一个复杂的重构任务，去吃午饭，在手机上批准权限请求，在公交上审查 diff。

深层含义是：**异步的人机协作**。Agent 持续工作，人类在方便的时候介入。这是所有编程工具的未来方向——Codex 已经在云端运行，但 Anthropic 的方案保持了本地会话（你的机器、你的文件系统），只把控制层搬到了手机。预计几个月内所有主流编程工具都会跟进类似功能。
