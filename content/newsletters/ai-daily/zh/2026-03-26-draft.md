## 🎓 概念科普

### 什么是 Agent Harness（Agent 工作台）？

你可能经常看到「Agent」这个词，但很少有人提到 Agent 跑在什么上面。Harness（直译是「挽具」）就是 Agent 运行时的支撑系统——它控制 Agent 能用什么工具、怎么拿到上下文、执行结果怎么反馈回来。

打个比方：Agent 是赛车手，Harness 是赛车 + 赛道 + 规则。同一个车手换一台车，圈速可能差好几秒。

Anthropic 今天的博客就证明了这一点：harness 设计的好坏对 Agent 表现的影响，有时比换一个更强的模型还大。这也是为什么 Claude Code 的 skills、hooks、CLAUDE.md 这些配置机制这么重要——它们本质上就是在调你的 harness。
