## 🎓 名词解读：什么是 Agent Harness？

今天多条新闻都提到了"Agent Harness"（Agent 挽具/框架）。简单说，Harness 就是包裹在 LLM 外面的脚手架——它把一个聊天机器人变成一个能"做事"的 Agent。

Harness 负责 LLM 自己做不了的事：调用工具、跨上下文窗口管理记忆、失败时重试、判断什么时候该停。你可以把 LLM 想成大脑，Harness 就是神经系统加骨骼。一个厉害的模型配上烂 Harness，表现反而不如普通模型配好 Harness。这就是为什么 Anthropic 今天那篇"长时间运行 Agent 的有效 Harness 设计"如此重要——瓶颈越来越多地出现在 Harness 上，而不是模型本身。

## 🎯 今日之选：Karpathy 的 AutoResearch

Andrej Karpathy 每次开源项目都值得关注——不是因为名人效应，而是他放出来的东西经常变成行业标配（nanoGPT、minGPT、micrograd）。AutoResearch 让 AI Agent 在单 GPU 上自主完成研究闭环：提出假设、写实验代码、跑实验、分析数据、迭代优化。

这件事之所以重要，是因为它是第一个靠谱的开源尝试去"关闭整个研究循环"——不只是写综述或查文献，而是真正地跑实验。如果这个模式跑通了，从"我想验证一个想法"到"数据告诉我答案"的时间会从几天缩短到几小时。而且它设计为在你已有的硬件上运行。
[查看详情 →](https://github.com/karpathy/autoresearch)
