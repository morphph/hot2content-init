## 🎓 概念科普：什么是 Tool Search（工具搜索）？

**Tool Search** 是 OpenAI Responses API 的新能力。传统做法是把所有可用工具的定义一股脑塞进上下文——浪费 token、拖慢速度、影响缓存。Tool Search 让模型在运行时按需发现和加载工具，类似于「懒加载」：模型先看一个轻量索引，选出需要的工具，再加载完整定义。当你有几十上百个工具时特别有用。

## 🎯 今日精选：AI 评测中的基础设施噪声问题

Anthropic 这篇关于基础设施噪声的文章值得仔细读。他们发现**同一个模型**仅因基础设施配置不同，评测分数就能波动好几个百分点。而排行榜上顶级模型之间的差距通常只有 1-3%，这意味着基础设施变量可能**比模型本身的性能差异还大**。网络延迟、沙箱配置、超时设置、重试逻辑这些因素都在叠加。对于做 coding agent 或者依赖 benchmark 选模型的团队来说，这是必读材料。

[查看详情 →](https://www.anthropic.com/engineering/infrastructure-noise)
