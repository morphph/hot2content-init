# 并行 Claude 造出 C 编译器，OpenClaw 生态一夜冒出十几个变种

Anthropic 工程博客连发三篇重磅文章，其中最炸的一篇：用一群 Opus 4.6 Agent 并行协作，从零写出了一个 C 编译器——然后人类基本没插手。与此同时，GitHub Trending 被 OpenClaw 生态刷屏了，nanobot、nanoclaw、picoclaw、zeroclaw……名字都快不够用了。

今天：Agent 自主开发的新里程碑、评测基准的「噪声」问题、以及 OpenClaw 技能生态的寒武纪大爆发。

## 🎓 概念科普：Agent Skills（技能系统）

**Agent Skills** 是让 AI Agent 获得新能力的模块化插件。你可以把它理解成手机 App Store——Agent 本身是操作系统，Skills 是你给它装的应用。

比如一个「写邮件」技能教会 Agent 如何调用 Gmail API 发送邮件，一个「数据分析」技能让它能连接数据库跑 SQL 查询。技能可以由社区贡献、相互组合，形成越来越强的能力链。

今天 GitHub 上 5400+ OpenClaw 技能的爆发式增长说明了一个趋势：未来 AI Agent 的竞争力不只取决于底层模型有多强，也取决于它能接入多少高质量技能。这和智能手机的发展路径几乎一模一样。

## 🎯 今日精选：Anthropic 用 Agent 团队从零造出 C 编译器
