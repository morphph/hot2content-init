## 🎓 概念科普：提示词缓存

**是什么：** 当你向 LLM API 发送请求时，系统可以缓存请求的开头部分（"前缀"）。如果下一个请求开头相同，就复用缓存的计算结果。

**为什么重要：** 像 Claude Code 这样的 Agent 一个会话中会发送几十次 API 调用，每次都共享相同的系统提示 + 工具 + 对话历史。没有缓存时，每次调用都从头处理所有内容。有了缓存，只有新增部分需要全价计算。

**关键点：** 它是前缀匹配——从头开始所有内容必须完全相同。改一个字符？缓存失效。调换工具顺序？缓存失效。所以 Anthropic 推出了自动缓存——让你不再需要操心缓存断点的问题。
