**2026 年 2 月 18 日**

## 🎓 概念科普：混合专家模型（MoE）

Qwen3.5-397B-A17B 使用 MoE——这是什么意思？

普通模型对每个 token 都激活**所有**参数。MoE 模型有很多"专家"子网络，但每个 token 只激活其中几个。Qwen3.5 总共有 3970 亿参数，但每个 token 只使用 170 亿（"A17B" 的含义）。

**为什么重要：** 你获得了 3970 亿参数模型的知识，但只花费 170 亿参数模型的计算成本。就像团队里有 20 个专家待命，但每个问题只咨询 2-3 个。

## 🎯 今日精选：Sonnet 4.6 进入 Claude Code

真正的故事不只是又一个模型发布——而是采用曲线。Claude Code 的周活跃用户自 1 月以来翻倍。Spotify 工程师停止了手动写代码。非技术团队在构建内部工具。现在 Sonnet 4.6 让这一切变得更便宜、更快。

如果你还没试过 Claude Code，今天就是时候。如果已经在用了，切换到 Sonnet 4.6，看看它是否能以 Opus 成本的零头处理你的工作流。
