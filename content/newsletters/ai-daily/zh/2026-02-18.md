# Sonnet 4.6 正式入驻 Claude Code，开源模型洪流涌入 HuggingFace

**2026 年 2 月 18 日**

Anthropic 的 Sonnet 4.6 今天登陆 Claude Code——接近 Opus 智能水平，成本却只是零头。与此同时，Moonshot、NVIDIA 和智谱 AI 的开源模型在 HuggingFace 上掀起一波热潮。闭源和开源模型之间的差距持续缩小。

## 🧠 模型

**Sonnet 4.6 已在 Claude Code 中上线** — Anthropic 的 Boris Cherny 确认其成本低于 Opus 4.6，但接近 Opus 级智能。早期测试显示开发者在编码任务上往往更偏好它。能力、速度和 token 效率之间的绝佳平衡。

- **GLM-5**（智谱 AI）— 1.3K ❤️。GLM 系列新一代。

## 🎓 模型小课堂：混合专家（MoE）

Qwen3.5-397B-A17B 使用了 MoE——这是什么意思？

普通模型对每个 token 激活**所有**参数。MoE 模型拥有多个"专家"子网络，但每个 token 只激活其中几个。Qwen3.5 总共有 397B 参数，但每个 token 只用 17B（即"A17B"的含义）。

**为什么重要：** 你能获得 397B 模型的知识量，却只付出 17B 模型的计算成本。就像有 20 个专家随时待命，但每个问题只咨询 2-3 个。
