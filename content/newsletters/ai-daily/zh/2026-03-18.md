## 🎓 概念科普：什么是 Agent Skills？

最近你可能频繁看到"Skills"这个词出现在 Agent 工具的描述里。Agent Skill 本质上是一段可复用的能力描述——告诉 Agent "你能做什么、怎么做"的配置文件。

打个比方：如果 Agent 是一个新员工，Skill 就是给他的工作手册。一个 SEO Skill 会告诉 Agent 怎么分析关键词密度，一个 Code Review Skill 会教它怎么检查代码质量。Skills 可以共享、组合、叠加，就像给 Agent 装不同的"技能模块"。

这和传统插件的区别在于：插件是硬编码的功能，Skill 更多是自然语言描述的行为指南，Agent 自己判断什么时候用、怎么用。今天 GitHub 上出现的各种 awesome-skills 列表（5400+ 个技能！），说明这个生态已经进入了爆发期。

## 🎯 今日精选：Anthropic Cowork — Agent 工作方式走出编程圈

Cowork 是 Anthropic 今天发布的新产品，核心理念是让非程序员也能像开发者用 Claude Code 一样完成工作——不是聊天式的问答，而是 Agent 式的任务执行。

为什么这条最重要？因为它代表了一个关键转折：过去一年 Agent 工具几乎全部集中在编码场景（Claude Code、Cursor、Codex），但知识工作中 90% 的任务不是写代码。Cowork 把"给 Agent 一个任务，让它自主完成"这个模式推广到写报告、做分析、整理数据等场景。同一天 GitHub 上就出现了开源替代品 openwork，说明这个方向击中了真实需求。

如果 Claude Code 改变了程序员的工作方式，Cowork 想改变所有人的工作方式。这是一个比编码 Agent 大得多的市场。
