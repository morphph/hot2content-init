## 🎓 模型小课堂

**什么是 AI 编码 agent 的"Plan Mode"？**
当你给 AI 编码 agent 一个复杂任务时，它可以直接动手（有风险——可能走错方向），也可以先停下来想想。Plan Mode 就是第二种方式：agent 分析你的请求，问澄清问题，展示带选项的实施计划，然后才开始写代码。Anthropic 刚在 Claude Code 的 Slack 应用中发布了这个功能。原理和好的人类工程一样——量两次，切一刀。这减少了无用功，让你有机会在 agent 花费大量 token 走错方向之前纠偏。
