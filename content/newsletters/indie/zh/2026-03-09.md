## 🏆 封面故事：Cursor 三个月翻倍，年化收入突破 $20 亿

这周最炸裂的数字：**Cursor 的年化收入在 2 月突破了 $20 亿**，三个月内翻了一倍。Bloomberg 独家报道，60% 收入来自企业客户。

去年 11 月 Accel 和 Coatue 领投的 $23 亿融资给了它 $293 亿估值，当时觉得疯狂，现在看收入增速……好像也没那么疯。

但隐忧也在：Anthropic 自己的 Claude Code 越来越强，Claude Code 创始人 Boris Cherny 甚至说"软件工程师"这个头衔 2026 年可能开始消失。上游吃下游，这戏码大家都懂。

整个 vibe coding 赛道的估值也在飙：**Lovable 一年内估值到 $66 亿，Replit 以 $90 亿估值融资**。AI 写代码不再是 demo，是真金白银的大生意。

## 💰 收入里程碑

### Emergent AI：双胞胎兄弟的 Vibe Coding 应用，6 个月做到 $30K MRR

两个双胞胎创建的 **Emergent AI: Vibe Code Apps**，一个 AI 辅助应用构建工具，上线半年爬到应用商店 #12 名，30K 下载，**月收入 $30K**。

更有趣的是获客方式——冰岛最大的喜剧网红 Greipur Hjaltason（TikTok 88 万粉）开始推广这个 app，假装是自己做的。网红 + vibe coding，这个组合拳值得关注。

🔗 [Social Growth Engineers](https://www.socialgrowthengineers.com/6-month-old-vibe-coding-app-hits-12-30k-mrr)

### "David"：40 岁零基础，AI 造 App 月入 $12K

一个 40 岁左右、完全没有开发经验的人，看到 TikTok 上有人用 AI 做 app 走红，自己也开干。一个月做到 $5K/月，现在已经超过 **$12K/月**。

不会写代码不是障碍，会找对工具、验证对需求才是核心能力。

## 🛠 Build Story

### Jan Luca Sandmann：不拿 VC，Bootstrap 做 AI Computer Agent

Jan Luca Sandmann 写了一篇很实在的文章，记录自己在 2026 年不融资、自力更生做 [Computer Agents](https://computer-agents.com) 的经历。

核心观点：2026 年 VC 的钱集中流向已验证的 AI 大项目和 Pre-IPO 巨头，早期创始人面临更长的融资周期和更高的 traction 门槛。**但 bootstrap 的优势是：你不需要讨好投资人，只需要讨好用户。**

他的产品方向是 computer-use agent（让 AI 操作真实电脑完成任务），跟 Anthropic 刚收购的 Vercept 是同赛道。

🔗 [Medium](https://medium.com/@jan.luca.sandmann/bootstrapping-an-ai-startup-in-2026-how-im-building-computer-agents-without-vc-in-a-selective-734284b40f88) | [DEV.to](https://dev.to/jan_lucasandmann_bb9257c/claude-code-to-ai-os-blueprint-skills-hooks-agents-mcp-setup-in-2026-46gg)

## 🎯 Takeaway

**这周的核心叙事：Vibe Coding 已经从"玩具"变成了"产业"。**

- Cursor $2B ARR 证明开发者愿意为 AI coding 付真金白银
- 双胞胎兄弟、40 岁零基础大叔的故事说明门槛在不断降低
- Anthropic 收购 Vercept、发布劳动力报告——AI 对工作方式的重塑已经从预测进入实测
- Solo founder + AI 的组合正在从"能做"进化到"做得好、做得赚钱"

**对独立开发者的建议：**
1. 别再观望了，选一个 vibe coding 工具开始你的第一个项目
2. 关注 distribution（获客）比关注 code quality 更重要——网红推广、TikTok 获客都是有效路径
3. Micro-SaaS 才是 indie hacker 的最佳起点，$5K MRR 就是一个改变生活的数字
