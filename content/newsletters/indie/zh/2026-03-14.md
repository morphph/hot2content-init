## 🏆 封面故事：Google Antigravity 从免费到收费，开发者炸锅

本周最大的 drama 来自 Google。Antigravity——Google 的 AI 编程工具——宣布调整定价方案，从之前"预览期免费"转向 AI Credit 体系（$25/2500 credits）。Pro 用户发现配额不再每隔几小时刷新，而是改成每周重置，项目直接卡住。

Reddit 和 Google 开发者论坛上一片怨声载道。有用户吐槽："之前免费让你上瘾，现在把付费按钮怼到你面前。"

**对独立开发者的启示：** 免费工具永远有定价风险。选技术栈的时候，别只看"现在免费"，要想清楚 vendor lock-in 的成本。

📎 [The Register 报道](https://www.theregister.com/2026/03/12/users_protest_as_google_antigravity/) · [Techzine 分析](https://www.techzine.eu/news/devops/139547/developers-complain-about-google-antigravity-pricing-structure/) · [Google 论坛讨论](https://discuss.ai.google.dev/t/looks-like-there-is-new-pricing-etc-for-quota-limits-check-it-out/130758)

## 💰 收入里程碑

### saas.group 突破 $100M ARR
德国科隆的 saas.group 本周宣布旗下 25+ 个 SaaS 品牌组合 ARR 突破 $1 亿，目标今年达到 $1.25 亿。他们的模式很有意思：专门收购已经盈利的 bootstrapped SaaS（通常 $2M-$10M ARR），然后用 AI 做整合升级。旗下品牌包括 DashThis、Rewardful、Ayrshare 等。

对于想 exit 的独立开发者来说，这是个好消息——有人在大规模收购盈利的小 SaaS。

📎 [TechEdge AI](https://techedgeai.com/saas-group-surpasses-100m-arr-as-it-eyes-further-acquisitions-to-drive-ai-transformations/) · [Pulse2](https://pulse2.com/saas-group-100-million-arr/)

### AI-native SaaS 估值溢价 1-3x
Livmo 的最新报告显示：AI-native SaaS 在 2026 年获得比传统 SaaS 高 1-3 倍的估值倍数，87% 的买家预期这个差距还会继续扩大。Bootstrapped SaaS 中位估值为 4.8x ARR（VC-backed 为 5.3x），差距在缩小。

📎 [Livmo 报告](https://livmo.com/blog/ai-impact-saas-valuations-2026/) · [SaaS 估值数据](https://livmo.com/blog/saas-valuation-multiples-2026/)

## 🛠 Build Story：Vibe Coding 2026——工具军备竞赛白热化

本周 LogRocket 发布了一份超详尽的 [AI 开发工具 Power Rankings](https://blog.logrocket.com/ai-dev-tool-power-rankings/)，对比 18 个 AI 模型 + 11 个开发工具，几个要点：

**Cursor** 2 月推出 Long-running Agents，可以把任务丢到远程机器跑 10 分钟以上，然后从编辑器/手机/Web 端查看结果。从"本地 IDE"变成"分布式 AI 工作站"。

**Claude Code** 强化了 Auto-Accept Mode 和 headless/SDK 工作流，更适合无人值守的长时间运行。WCAG 合规检查和浏览器兼容性测试也是亮点。

**OpenAI Codex** 桌面端支持了 Windows，GPT-5.4 带来 100 万 token 上下文 + 原生 computer use。

**Google Antigravity** 主打 AI Studio 无缝集成（"Open in Antigravity" 一键从原型到应用），但定价问题见封面故事。

**Kimi Code** 和 **Gemini CLI** 是目前仅有的两个支持自托管的开源工具。

📎 [Kanaries 工具对比](https://docs.kanaries.net/articles/best-vibe-coding-tools) · [LogRocket Rankings](https://blog.logrocket.com/ai-dev-tool-power-rankings/)

## 📊 趋势信号

1. **"Revenue Hallucination" 成为热词。** Reuters 报道 Anthropic 法庭文件显示其累计总收入"超过 $50 亿"，但宣传的年化 run rate 是 $190 亿。OpenAI 年化收入达 $250 亿。大模型公司收入增长惊人，但 run rate ≠ 实际累计收入，投资人越来越关注这个差距。

2. **Vibe Coding 有了自己的 Wikipedia 页面。** 从 Andrej Karpathy 的推文到学术论文、企业会议（Salesforce TDX 2026 有专门的 Vibe Coding 赛道），这个概念已经完全进入主流。连 Linus Torvalds 都用 Antigravity vibe code 了一个工具组件。

3. **Solopreneur Tech Stack 2026：** 越来越多独立开发者选择 Hetzner VPS（性价比碾压 Vercel serverless）+ 轻量 AI 工具，而不是堆叠昂贵的云服务。$500/月的 AWS 账单正在被 $10/月的 VPS 替代。

4. **AI Design Systems 会议（3/19-20）** 将聚焦 AI Design + Vibe Coding + MCP 集成，有 WhatsApp、Miro、Indeed 的 speaker。

## 🎯 Takeaway

**本周的核心教训：免费午餐终会结束。**

Google Antigravity 从免费到收费的转变是个完美的案例研究。对独立开发者来说，2026 年选技术栈的原则越来越清楚：

- **能自托管的优先**（Kimi Code、Gemini CLI）
- **有明确定价模型的比"暂时免费"的更值得信赖**
- **AI-native 是加分项**（1-3x 估值溢价说明市场认可）
- **Prompt 是资产**，该用工程方式管理了

如果你正在做一个 bootstrapped SaaS，好消息是：像 saas.group 这样的买家在积极收购 $2M-$10M ARR 的盈利产品。把产品做好、做盈利，exit 渠道比以前多了。
