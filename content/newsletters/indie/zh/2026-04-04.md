## 🏆 封面故事：Claude Code 源码大泄露——2026 年最魔幻的"开源"事件

3 月 31 日，安全研究员 Chaofan Shou 发现 Anthropic 的 Claude Code CLI 工具在 npm 包中意外携带了一个 60MB 的 source-map 文件（`cli.js.map`），直接暴露了 **512,000 行 TypeScript 源码和近 2,000 个内部文件**。

泄露内容令人震惊：
- 完整的 agentic workflow 架构和 tool-calling 实现
- 一个被称为 "Buddy/companion" 的系统，计划在 4/1-4/7 上线
- 源码中发现 Anthropic 使用 Claude Code 做"隐身"开源贡献的 system prompt
- 泄露的 repo 在 GitHub 上迅速成为**史上增长最快的仓库之一**

更魔幻的是，这距离 Anthropic 上一次泄露（Claude Mythos 模型文档）仅隔 5 天。有人怀疑这是史上最牛的 PR stunt，但 Anthropic 确认是 npm 打包错误。

**独立开发者启示：** 泄露的架构是一份价值连城的 AI agent 工程教材。如果你在做 coding agent 相关的产品，必读。

## 🛠 Build Story

### Crossnode：把 AI Agent 变成付费产品，5 分钟搞定
Product Hunt 本周新秀。核心理念：你在 n8n 里做好了一个 AI automation workflow？用 Crossnode 一键包装成带登录、计费、用量限制的 SaaS 产品。不用写后端，不用接 Stripe。

对独立开发者来说，这可能是"AI 工作流变现"最短的路径。

🔗 [Product Hunt](https://www.producthunt.com/products/crossnode)

### Baton：AI Coding Agent 的指挥中心
另一个值得关注的新工具——Baton 是一个桌面 App，让你同时运行 Claude Code、Codex、OpenCode 等多个 terminal-based agent，每个都在独立的 git worktree 里工作，互不干扰。

Agent 编排正在从"大厂特权"变成"独立开发者标配"。

🔗 [Product Hunt](https://www.producthunt.com/products/baton-2)

## ⚡ Stack Watch

### Google Gemma 4 发布：真正的开源 + Apache 2.0
4 月 3 日，Google 发布 Gemma 4 系列开源模型：
- **四个尺寸：** E2B、E4B、26B MoE、31B Dense
- **重大变化：** 正式切换到 Apache 2.0 许可证

对独立开发者的意义：终于有一个真正开源、可商用、能跑 agentic 任务的小模型家族了。

## 📊 趋势信号

1. **Agent 编排 > 单 Agent 能力。** Cursor 3、Baton、Superset、Vibe Kanban——本周至少 4 个产品在做"多 agent 并行"。这不是巧合，是信号。

2. **"Vibe coding" 正在分化。** 入门者用 Lovable/Bolt 快速出 MVP，进阶者用 Cursor + Claude Code 做复杂项目，专业团队开始走"PRD first → agent execute"路线。工具链正在按能力分层。

3. **开源 AI 进入实用区。** Gemma 4 的 Apache 2.0 + 端侧可用 + agentic 能力，让独立开发者终于可以认真考虑"不依赖 API，本地跑 agent"。

4. **AI 工作流变现基础设施出现。** Crossnode 这类"workflow → paid SaaS"的工具暗示一个趋势：未来独立开发者的产品可能不是 App，而是一个精心调教的 AI workflow。

5. **Anthropic 的安全问题。** 一周内两次重大泄露（Mythos 文档 + Claude Code 源码），对一个以安全为品牌核心的公司来说，这是一个值得关注的信号。

## 🎯 Takeaway

本周的核心主题只有一个：**从"AI 写代码"到"AI agent 当员工"的范式迁移正在加速。**

Cursor 3 不再把自己定位为 code editor，而是 agent orchestrator。Baton 和 Superset 在做同样的事。Claude Code 的泄露源码展示了 Anthropic 如何构建 agent scaffolding。Google Gemma 4 让你本地跑 agentic model。

如果你是独立开发者，现在最值得投资的技能不是"如何写更好的 prompt"，而是：
- 如何设计和编排多个 AI agent 协同工作
- 如何把 AI workflow 打包成可计费的产品
- 如何在"全自动"和"人类监督"之间找到正确的平衡点
