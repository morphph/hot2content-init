## üéì MODEL LITERACY

**What is an "agent skill"?**  
A skill is a reusable bundle of instructions, scripts, and resources that tells an AI coding agent how to complete a specific task ‚Äî like running tests, deploying code, or analyzing security vulnerabilities. Instead of explaining the same workflow every time, you package it once as a skill. Now that skills are an open standard, the same skill file works across Claude Code, OpenAI Codex, and other agent platforms.
