## üéì MODEL LITERACY

**What is server-side compaction?** When AI agents work on long tasks, they accumulate context ‚Äî every file read, every tool call, every response. Eventually they hit the model's context limit and forget earlier work. Server-side compaction is OpenAI's solution: the server automatically summarizes and compresses older context, letting agents run for hours instead of minutes without losing critical information. Think of it as giving the agent a notebook it can review instead of relying purely on short-term memory.
