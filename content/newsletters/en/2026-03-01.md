## 🎓 MODEL LITERACY

**What is MCP (Model Context Protocol)?** MCP is a standard that lets AI models connect to external tools and data sources through a unified interface. Think of it like USB for AI — instead of custom integrations for every service, tools speak a common protocol. When Apple adds "MCP support" to Xcode, it means any MCP-compatible tool can plug into the IDE without Apple writing specific code for each one.
