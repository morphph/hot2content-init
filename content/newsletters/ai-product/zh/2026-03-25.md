## 🏆 封面故事：Anthropic 对战五角大楼，AI 公司与政府的边界之争

本周最炸裂的新闻不是哪家又出了新模型，而是 Anthropic 和美国政府正式在联邦法庭对簿公堂。

事情的起因：Anthropic 拒绝让 Claude 用于国内大规模监控和完全自主致命武器系统。特朗普总统直接下令所有联邦机构"立即停止"使用 Anthropic 的产品，国防部长 Hegseth 更是把 Anthropic 列为"供应链风险"——这个标签通常是留给外国对手的。

本周一（3月24日）庭审中，联邦法官 Lin 直接表示这三项行政措施"令人不安"（troubling），质疑五角大楼的做法是否与其声称的国家安全担忧相称："如果担心的是作战指挥链的完整性，直接停用 Claude 就行了，为什么要搞供应链风险认定？"

与此同时，军方内部使用者告诉路透社：Claude 被认为优于替代方案，六个月内完成替换"说起来容易做起来难"。

这不只是一家公司的事。这是 AI 公司能否对自己的技术用途说"不"的先例之战。

## 🔥 本周大事

### 1. OpenAI 发布 GPT-5.4 Mini 和 Nano

3月18日，OpenAI 正式推出 GPT-5.4 家族的两个效率型成员。GPT-5.4 Mini 直接对免费用户开放（通过 Thinking 功能），支持 **100 万 token 上下文窗口**、原生 computer use 和新的工具搜索机制。

定价方面，GPT-5.4 标准版 API 为 $2.50/$15（input/output per million tokens），Pro 版 $30/$180。Mini 和 Nano 显著更便宜，目标是高吞吐低延迟场景。

**对开发者的影响：** 这基本上是把接近旗舰级推理能力以极低成本推向生产环境。编码工作流提速明显。

### 2. Anthropic 产品周：Computer Use + Dispatch + Code Channels + Auto Mode

Anthropic 本周疯狂出货，一口气推了四个大更新：

- **Computer Use（预览版）**：Claude Cowork 和 Claude Code 现在可以直接操控你的电脑——点击、滚动、导航，像人一样操作屏幕完成任务。会在访问新应用前请求权限。
- **Claude Dispatch**（3/17）：从手机发指令给桌面端的 Claude。出门前安排任务，回来看结果。跨设备持久线程。
- **Claude Code Channels**：把 Claude Code 接入 Telegram 和 Discord，随时随地发消息让它干活。VentureBeat 直接称其为"OpenClaw 杀手"。
- **Auto Mode**（3/24）：Claude Code 新的权限模式，介于默认（每步确认）和绕过权限之间，让 AI 更自主但仍有安全护栏。

## 🎯 Takeaway

1. **AI 公司的政治风险成为现实。** Anthropic vs 五角大楼不只是个案——任何 AI 公司拒绝特定用途都可能面临政府报复。这会深刻影响其他公司的安全策略决策。

2. **"效率模型"成为新战场。** OpenAI 的 Mini/Nano、Google 的 Flash 系列——大家都在卷"用更小更便宜的模型做更多事"。对开发者来说是好消息：同样效果，成本可能降一个数量级。

3. **Anthropic 的 Agentic 产品力爆发。** Computer Use + Dispatch + Code Channels，一周之内把"AI agent 操控你的电脑"从概念变成了产品。这个速度很夸张。

4. **微软的 AI 焦虑溢出屏幕。** Copilot 重组、Suleyman 被调走、被迫撤回过度推送——$650M 挖来的 AI 负责人两年后被边缘化，信号很明确。

5. **三巨头 IPO 即将到来。** SpaceX + OpenAI + Anthropic 合计 $2.9 万亿潜在市值。2026 可能是科技 IPO 的超级年份。
