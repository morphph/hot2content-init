# OpenAI 发布企业级 Agent 平台，Anthropic 的 Opus 4.6 自己撸了个能跑的 C 编译器

企业 AI Agent 大战升级了。OpenAI 推出 Frontier——一个部署"能干实事的 AI 同事"的完整平台；而 Anthropic 那边轻描淡写地提了一嘴：他们的 Opus 4.6 Agent 团队花了两周时间，自主开发出了一个 C 编译器，现在已经能在 Linux 上跑了。与此同时，Qwen 扔出了一个 3970 亿参数的多模态 Agent 模型，ChatGPT 单日消息量突破 40 亿条。

**今日看点：** 企业级 Agent 平台正面对决，开源多模态模型迎头赶上，新研究质疑 AI 辅助编程是否真的帮助开发者成长。

## 🧠 模型

• **Qwen3.5 作为原生多模态 Agent 发布** — 阿里发布了一个 3970 亿参数的 MoE 模型（170 亿激活参数），原生支持对话、图像/视频理解、图像生成、文档处理、网页搜索和工具调用。在 Agentic AI 领域，这是一个相当能打的开源选手。(HN: 350 分, 171 评论) — Qwen Blog [阅读更多 →](https://qwen.ai/blog?id=qwen3.5)

• **GPT-5.3-Codex-Spark 上线，主打实时编程** — OpenAI 的新款超快模型专为实时编程打造，作为研究预览版向 ChatGPT Pro 用户开放，支持 Codex 应用、CLI 和 IDE 扩展。 — @OpenAIDevs [阅读更多 →](https://x.com/OpenAIDevs/status/2022009906329739681)

• **Voxtral Transcribe 2 发布，延迟低于 200ms** — Mistral 发布了新一代语音转文字模型，具备顶级转录能力、说话人分离功能，实时延迟低于 200ms。Whisper 的有力开源替代品。 — @MistralAI [阅读更多 →](https://x.com/MistralAI/status/2019068826097213953)

• **Qwen3.5-397B 权重上架 HuggingFace** — 完整模型权重现已可供自托管，本地大模型社区终于能用上前沿级 MoE 模型了。 — Reddit r/LocalLLaMA [阅读更多 →](https://www.reddit.com/r/LocalLLaMA/comments/1r656d7/qwen35397ba17b_is_out/)
