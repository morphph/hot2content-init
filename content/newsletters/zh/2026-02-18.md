## 🎓 概念科普

**为什么 100 万 token 的上下文窗口会改变模型的功能？**

上下文窗口是模型一次可以“看到”的文本量。凭借 100 万个 token（约 75 万个单词），Sonnet 4.6 可以在工作内存中保存整个代码库、整本书或数月的对话历史记录。这不仅仅是阅读更多内容——它还实现了像长程规划这样的涌现行为，而较短上下文的模型实际上无法执行这些行为，因为它们在到达终点之前就忘记了开头。

## 🎯 今日精选

**SkillsBench：自生成的 Agent 技能在很大程度上是无用的** — arXiv (94 HN pts, 48 comments)

这篇论文挑战了 Agent 开发中的一个核心假设：Agent 可以通过生成可重用的技能来引导自己的能力。该基准测试发现，这些自生成的技能无法跨任务转移。如果你正在构建依赖于技能积累的 Agent 系统，这项研究表明你可能需要人工策划的技能库。这个结论值得所有 Agent 开发者警惕。[查看详情 →](https://arxiv.org/abs/2602.12670)
